# BoyBarleyPanel

A lightweight yet powerful VPS control panel.

## Features

- System Monitoring
- Service Management
- File Manager
- Database Management
- Security Tools
- Backup System
- API Access

## Requirements

- PHP 7.4 or higher
- MySQL 5.7 or higher
- Nginx/Apache
- Linux-based OS (Ubuntu/Debian recommended)

## Installation

### Quick Install
```bash
wget -O install.sh https://raw.githubusercontent.com/yourusername/BoyBarleyPanel/main/installer/install.sh
chmod +x install.sh
./install.sh
